Quick Start Guide

Get up and running with TSEK in 4 simple steps.

1

Create a Class

Go to Classes → Add Class. Enter the course name and section code.

2

Add Students

Open the class → Add Student. Enter names and ID numbers.

3

Generate an Answer Sheet

Go to Generate Answer Sheet. Configure sections, set the answer key, and print.

4

Scan & Grade

Click TSEK NOW, scan the filled sheet, select the exam, and grade.
